# 어제이슈 v3 아이콘 / 문구 방향

> **[폐기됨 2026-06-12]** 소셜갭 캐릭터 아이콘 컨셉은 사용자 결정으로 폐기.
> 제출 아이콘은 인앱 브랜드 마크(상승 막대 + 떠오르는 해)와 동일한 디자인을 사용한다.
> 태그라인("어제 다들 얘기한 이슈를 오늘 아침 5분 만에")은 유지.

## 문제 정의

기존 추상 아이콘은 앱 취지와 연결이 약했다. 앱의 핵심 상황은 "다른 사람들은 어제 이슈를 이야기하는데, 나만 맥락을 모르는 느낌"이다.

## 새 아이콘 방향

- 장면: 두 명은 말풍선으로 어제 이슈를 이야기하고, 오른쪽 작은 캐릭터는 물음표와 함께 뒤처진다.
- 캐릭터: 대두, 단순한 얼굴, 둥근 몸, 귀여운 비율.
- 의미: 어제 화제를 놓친 사람도 오늘 아침 빠르게 따라잡게 해준다.
- 톤: 너무 유아적이지 않게, 검정 배경 + 파란 포인트로 앱 아이콘 느낌 유지.

## 추천 앱 설명

`어제 다들 얘기한 이슈를 오늘 아침 5분 만에`

## 대안 문구

1. `나만 몰랐던 어제의 이슈, 오늘 바로 따라잡기`
2. `어제의 화제, 오늘 아침에 맥락까지`
3. `다들 말한 그 이슈, 이제 나도 안다`
