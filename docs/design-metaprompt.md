# 어제 이슈 - 디자인 메타프롬프트

Figma(또는 다른 디자인 도구/AI)에서 화면을 재현·발전시킬 때 그대로 붙여 쓰는 명세.
현재 코드(app/src/styles.css, view.js)는 이 명세의 구현본이다.

---

## 메타프롬프트

> 한국의 모바일 뉴스 브리핑 앱 "어제 이슈"를 디자인하라.
> 사용자는 출근길에 1~3분 안에 어제 화제가 된 이슈 25개를 훑는다.
>
> **무드**: 신문의 신뢰감 + 토스의 가벼움. 종이 질감의 웜 그레이 배경(#F5F3EF) 위에
> 아이보리 카드(#FFFDFA). 차갑고 기계적인 순백/순회색 금지. 그라데이션·일러스트·
> 의미 없는 아이콘 금지. 콘텐츠(제목 텍스트)가 항상 주인공.
>
> **타이포**: Pretendard. 브랜드 타이틀 32/950, 리드 제목 26/950(자간 -0.75),
> 카드 제목 16.5/750, 메타 12.5/650. 한국어 word-break: keep-all.
>
> **컬러**: 잉크 #15171B, 보조 #4C535D, 뮤트 #8C929B.
> 카테고리 5색 - 정치 #D94841 / 경제 #16865F / 사회 #3268C7 / 문화 #7357CC / 연예 #D9821F.
> 카테고리 색은 라벨 칩과 해시태그에만 12% 틴트로 사용. 면적 큰 컬러 블록 금지.
>
> **구조**:
> - 상단: eyebrow(MORNING ISSUE DIGEST, 10px 자간 0.13em) + 브랜드 + 날짜 한 줄, 우측 저장 칩(북마크+카운트)
> - 탭: 스티키, 알약형, 활성 = 잉크 배경 흰 글자. blur 배경
> - 헤드라인 탭: 섹션 키커("오늘 먼저 볼 5개") → 리드 카드(제목+요약+주목도/민감도 배지) → 2~5위 행(순위 숫자+제목)
> - 카테고리 탭: 카드 5개(카테고리 칩 / 출처·상대시간 / 제목 / 한 줄 요약 / 주목도 배지+태그 2개)
> - 카드: radius 28, 1px 보더 헤어라인 + 종이 그림자(0 22px 70px -38px), 인셋 하이라이트 1px
> - 상세: 바텀시트(radius 16 상단), 핸들, 제목 → 주목도/민감도 메트릭 카드 2개 →
>   "무슨 일?" / "왜 중요해?" / "핵심 포인트(불릿)" 블록(카테고리색 라벨) → 태그 → 저장+원문 CTA(잉크 배경)
> - 모션: 0.48s cubic-bezier(.16,1,.3,1), 탭/카드 active scale(.985). 과한 애니메이션 금지
>
> **상태**: 스켈레톤(시머), 오류(재시도 버튼), 빈 저장함, 갱신 지연 배너(웜 옐로 보더).
>
> **금지**: 말줄임(...) 텍스트 잘림, 카드 안 카드 남발, 의미 없는 큰 그래픽,
> 파스텔 무지개 칩 남발, 영문 위주 UI.

## Figma 연동 메모

이 환경에는 Figma MCP/플러그인이 연결돼 있지 않다 (레지스트리 검색 결과 없음, 2026-06-11).
Figma에서 작업하려면:
1. Figma 데스크톱 앱 → Dev Mode MCP 서버 활성화 (Preferences → Enable Dev Mode MCP Server)
2. Claude Code에 `claude mcp add --transport sse figma-dev-mode-mcp-server http://127.0.0.1:3845/sse` 로 등록
3. 이후 "Figma 프레임 → 코드 반영" / "코드 → 디자인 토큰 추출" 왕복 작업 가능
